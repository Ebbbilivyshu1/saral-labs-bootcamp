# Task 1: GPT Timeline Recreation

## Overview
This project recreates a visual timeline of GPT (Generative Pre-trained Transformer) research papers as an interactive web page using HTML, CSS, and JavaScript.

The goal was to extract information from an image using OCR and present it in a structured, visually engaging, and interactive format.

---

## Live Demo
👉 https://github.com/Ebbbilivyshu1/saral-labs-bootcamp

---

## Features
- Interactive timeline layout
- Displays:
  - Year
  - Paper title
  - Short description
- Hover / click effects for better user experience
- Clean and responsive design

---

## Tech Stack
- HTML
- CSS
- JavaScript

---

## OCR Approach
- Extracted text from the provided timeline image
- Used:
  - Google Docs OCR (primary method for accuracy)
  - Tesseract OCR (experimented in Colab)
- Cleaned and structured the extracted data manually
